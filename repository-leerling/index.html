Repository Dries Van Ransbeeck

Structuur

De repository is helemaal niet gestrucutureerd. Alles wat hij voor school heeft gedaan zit gewoon los in de map. Er is een map die img heet, en daarin zitten er foto's maar die staan er ook zonder de map. De foto's staan eigenlijk dubbel.
De map heet ook "les 1" i.p.v. "MI1" of een andere logische naam. Hij heeft wel de opdracht voor de klant deftig in een map gezet.

Volledigheid

Het ziet er allemaal in orde uit.

Regelmatigheid

De leerling gaat gemiddeld elke week op github zijn werk uploaden.

Correctheid

Er zijn geen fouten in zijn code, alles is in orde.

Bereidheid tot verbetering

De leerling heeft niks anders geupload want de code was van de eerste keer juist.
